## Getting and Cleaning Data Project.

This repo contains script and codebook for Getting and Cleaning Data Course
project.

##run_analysis() script

The script contains one function run_analysis that perform the five tasks 
detailed in the [project page](https://class.coursera.org/getdata-016/human_grading/view/courses/973758/assessments/3/submissions).
The script is splitted in sections and the task actually performed is stated in
parenthesis within each section.

In order to work properly original data set must be saved in the working directory, folder name cannot be changed.

This script reads 8 files of raw data and returns the file 'results.txt', a tidy data set containing a summarized set of averages of mean and standard deviation values for each subject and activity performed.

A detailed explanation of the variables and summarizing process are included in the code book.
